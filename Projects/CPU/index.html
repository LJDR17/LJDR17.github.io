What is a CPU?

A central processing unit is the electronic circuitry within a computer that executes instructions that make up a computer program. The CPU performs basic arithmetic, logic, controlling, and input/output operations specified by the instructions in the program.

So how does it work?

The working of the CPU is a 3 step process. First, an instruction is fetched from memory. Second, the instruction is decoded. Third, the instruction is executed and an operation is performed. These three steps repeat in a cycle that begins again with the CPU fetching the next instruction. The steps are referred to as the fetch-decode-execute cycle.

The speed of a CPU is measured by how many cycles it executes per second, measured in GHz (gigahertz) - this is known as the clock speed. A CPU with a clock speed of 3.2 GHz executes 3.2 billion cycles per second.

Now that we know how it works, what's inside of it?

Well, there are many components that each have different functions, hover over each point to see its function:

  • Control unit

    The Control Unit is the component of the processor which directs the operations of the CPU. It controls and coordinates the activities of the CPU and manages the flow of data between the CPU and other devices. It also decodes instructions.

  • ALU

    The ALU is the Arithmetic Logic Unit which is where calculations are carried out such as mathematical tasks, logic tests and data comparisons.

  • Cache

    Cache is a small area of very fast memory within the CPU which is used to store data when carrying out instructions.

  • MAR

    The MAR is the Memory Address Register which holds the memory address of data or instructions that is to be read from or written to memory (RAM).

  • MDR
    The MDR is the Memory Data Register which temporarily stores data that has been read or data that needs to be written to memory.

  • Accumulator

    This is where results of calculations carried out by the ALU are temporarily stored until they are needed.

  • CIR

    The CIR is the Current Instruction Register and holds the instruction currently being executed.
